# ACT sheet — Multiple carboxylase deficiency (biotinidase / HLCS deficiency) (MCD)

**Urgency:** URGENT

Confirm within 1–2 weeks. Highly biotin-responsive — early biotin starts before any neurological/cutaneous manifestation and reverses biochemistry rapidly.

## Confirmatory testing

1. **Plasma biotinidase activity** — definitive for biotinidase deficiency (most common form); profound <10%, partial 10–30%.
2. **Urine organic acids** — 3-OH-isovaleric, 3-methylcrotonylglycine, methylcitric acid, lactic acid, propionylglycine (combined-carboxylase pattern).
3. **Plasma acylcarnitines** — C5-OH + C3 + lactic acid.
4. **HLCS sequencing** — for neonatal-onset / severe MCD with normal biotinidase activity.
5. **BTD molecular analysis** — confirms biotinidase deficiency.
6. **Biotin trial (5–10 mg/day)** — biochemical normalisation within days; therapeutic and confirmatory.

## Pitfalls and false-positive causes

1. **Isolated 3-MCC deficiency** — C5-OH elevated but biotinidase activity NORMAL.
2. **HMG-CoA lyase deficiency** — high C5-OH; distinguished by HMG in UOA.
3. **Maternal biotinidase deficiency** — affects newborn screen biochem; usually mild and self-resolving.
4. **Anti-epileptic medication (carbamazepine, phenytoin, valproate)** — induce biotinidase activity reduction; spurious low NBS.
5. **Premature infants** — biotinidase activity can be transiently low.
6. **TPN without biotin supplementation** — secondary functional deficiency.
7. **Heat-degraded DBS** — falsely-low biotinidase activity; resampling normal.
